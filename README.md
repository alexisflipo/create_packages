# INSTRUCTIONS
## Cloner le repo 
`git clone https://github.com/alexisflipo/create_packages.git`

## Aller à la racine du repo cloné 

**Installer via pip**

`pip install .`

## Sortir du repo

**Remplacer nom_du_dossier par le nom de votre choix**

`create_pkg nom_du_dossier`

**TADAM !!! Architecture créée**
